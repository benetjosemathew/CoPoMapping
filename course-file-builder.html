What are we doing today?

Course master data — CO–PO/PSO mapping export

Fastest path:
Get this file from the AES portal: go to aesajce.in/academic/comaster.php ↗, find your subject, and click its CO→PO/PSO column to download the mapping file. (Requires AES login — the link only opens if you're already signed in.) Uploading it here auto-fills course code, course name, all COs, the CO–PO and CO–PSO matrices, and the full PO/PSO wording.

Don't have portal access, or teaching another department? Leave this blank and fill course details, COs and the mapping manually further down.

Course details

Subject name & code auto-fill from the AES export above if uploaded. Class/batch and term aren't in that file — the portal export is course-level, not offering-specific — so enter those here.

Choose 12 POs only if your course file still follows the older SAR format. In legacy mode, old PO7 (Environment & Sustainability) has no published PIs — mappings to it are justified by direct linkage to the PO statement, like PSOs.

KTU-affiliated batches typically can't claim granular end-sem CO coverage since the university doesn't publish a question-to-CO map. Autonomous courses set their own end-sem paper, so genuine per-CO coverage commentary is possible. Auto-detect checks the uploaded workbook for the "ALL COs" tag and tells you what it found — override here if it guesses wrong.

Previous semester's CO Attainment workbook

Upload the standard 5-sheet workbook (Assessment Methods · Students CO Score · Indirect Assessment · Direct Assessment · CO Attainment). Only the two summary sheets are read — no student names or individual marks are extracted or leave this page.

Assessment methods & project component for the next time this course runs

This describes what's planned going forward — not necessarily what was used in a past offering. For a new course, that's simply this offering; for a review, it's what you intend to use next cycle (carry forward, drop, or add methods as you see fit).

A CO6 (teamwork) is added only for a full project or mini-project. Minor group assignments do not honestly justify PO8/PO9 mapping.

Type A (default for every course): Two internal series examinations. These are assumed automatically — nothing to tick.
